# 角色权限与页面状态规范

## 角色

| 角色              | 识别来源                                                                   | 可见能力                                         | 受限能力                                               |
| ----------------- | -------------------------------------------------------------------------- | ------------------------------------------------ | ------------------------------------------------------ |
| 游客/未登录用户   | 无有效用户信息                                                             | 浏览首页、服务商城、书苑、公开资源/活动/商机列表 | 提交需求、报名、购买、发布商机、申请接单、查看个人记录 |
| 普通用户          | `getUserInfo` 或 `getUserProfile` 返回基础用户信息                         | 浏览公开内容、维护个人资料、发起部分咨询或报名   | 会员价、部分权益、认证机构动作、后台页面               |
| 会员用户          | `vip_level` 或 `vip_level_text` 有值                                       | 查看会员权益、会员价、会员专属入口               | 仍需认证的企业动作                                     |
| 认证机构          | `certification_status` 为已通过，或 `certification_status_text` 表达已认证 | 发布商机、提交企业需求、展示企业身份             | 后台审核权限                                           |
| 服务商/接单方     | 由接口角色字段或后续权限接口确认                                           | 申请接单、查看申请记录                           | 平台后台处理动作                                       |
| 后台运营/审核人员 | `role`、`role_text` 或后台接口权限确认                                     | 核销、认证审核、订单确认、资源需求处理、商机撮合 | 普通用户前台权益不自动继承                             |

## 通用页面状态

| 状态   | 触发条件                         | 推荐标题   | 推荐说明               |
| ------ | -------------------------------- | ---------- | ---------------------- |
| 加载中 | 接口请求未完成                   | 加载中     | 正在获取最新数据       |
| 空态   | 接口成功但列表为空或核心数据为空 | 暂无数据   | 当前没有可展示的记录   |
| 错误   | 接口失败或解析失败               | 加载失败   | 请稍后重试             |
| 无权限 | 当前用户角色不足                 | 暂无权限   | 当前账号不能访问该功能 |
| 未登录 | 需要登录但没有用户信息           | 请先登录   | 登录后可继续操作       |
| 认证中 | 认证资料已提交但未通过           | 认证审核中 | 平台正在审核企业资料   |
| 审核中 | 业务单据已提交等待后台处理       | 审核中     | 平台将在处理后更新状态 |
| 已失效 | 活动、订单、权益或商机过期       | 已失效     | 该记录已过有效期       |
| 已取消 | 用户或平台取消记录               | 已取消     | 该记录已取消           |
| 已完成 | 流程完成                         | 已完成     | 当前流程已处理完成     |

## 文案规则

- 核心字段缺失时，优先解释真实原因，不补假内容。
- 后台页面空态使用“暂无待处理记录”，避免暗示系统异常。
- 会员权益缺失使用“暂无可展示的会员权益”。
- 活动核销信息缺失使用“该活动暂未配置核销信息”。
- 订单支付状态缺失使用“支付状态确认中”。
- 商机撮合状态缺失使用“平台暂未返回撮合状态”。

## 业务状态建议

以下状态用于页面展示和后续接口对齐。当前接口未返回对应状态时，页面只展示真实可判断的状态，不通过前端硬编码制造业务结果。

### 订单与支付

| 状态       | 含义                               | 用户侧推荐动作               | 后台侧推荐动作     |
| ---------- | ---------------------------------- | ---------------------------- | ------------------ |
| 待支付     | 订单已创建，尚未发起或完成支付     | 继续支付、取消订单           | 不处理             |
| 待上传凭证 | 用户选择对公转账，但未上传凭证     | 上传凭证、取消订单           | 不处理             |
| 待财务确认 | 凭证已提交，等待财务确认           | 查看订单状态                 | 确认到账、驳回凭证 |
| 已驳回     | 凭证或支付信息不符合要求           | 查看原因、重新上传或联系客服 | 查看处理记录       |
| 已支付     | 支付成功，但业务权益可能未完全生效 | 查看后续服务状态             | 开通权益或推进交付 |
| 已完成     | 订单对应服务或权益已处理完成       | 查看权益、服务或评价         | 查看记录           |
| 已取消     | 用户或平台取消订单                 | 重新下单                     | 查看记录           |

接口缺口：当前需要订单状态枚举说明、后台确认到账接口、后台驳回凭证接口，以及凭证与订单状态绑定的结果字段。

### 会员开通与权益

| 状态       | 含义                         | 推荐展示                   |
| ---------- | ---------------------------- | -------------------------- |
| 未开通     | 用户没有有效会员             | 展示会员方案和开通入口     |
| 开通待确认 | 已提交会员订单或凭证，待处理 | 展示等待确认说明           |
| 已生效     | 会员等级和权益已生效         | 展示等级、有效期和权益     |
| 已过期     | 会员有效期结束               | 展示续费入口               |
| 开通失败   | 支付或审核未通过             | 展示失败原因和重新处理入口 |

接口缺口：当前需要会员生效结果、有效期、权益使用明细、剩余额度和开通失败原因。

### 企业认证

| 状态       | 含义                       | 推荐展示                   |
| ---------- | -------------------------- | -------------------------- |
| 未提交     | 用户尚未提交认证资料       | 展示认证表单               |
| 审核中     | 资料已提交，等待平台审核   | 展示提交信息和等待说明     |
| 已通过     | 企业身份认证通过           | 展示认证身份和相关能力入口 |
| 已驳回     | 审核未通过                 | 展示驳回原因和重新提交入口 |
| 需补充材料 | 审核要求补充证照或企业资料 | 展示补充材料要求           |

接口缺口：当前需要后台认证申请列表、认证详情、通过、驳回和补充材料接口。

### 活动报名与票务

| 状态   | 含义                         | 推荐展示           |
| ------ | ---------------------------- | ------------------ |
| 待支付 | 付费报名已提交但未完成支付   | 展示继续支付入口   |
| 待确认 | 对公转账或人工确认报名中     | 展示等待确认说明   |
| 已报名 | 报名成功，可生成或查看电子票 | 展示电子票入口     |
| 已核销 | 现场已扫码签到               | 展示核销时间和状态 |
| 已取消 | 报名已取消                   | 展示取消状态       |
| 已过期 | 活动已结束或票已失效         | 展示不可使用说明   |

接口缺口：当前需要付费报名支付状态、电子票生成条件和拼团状态接口。

### 商机撮合

| 状态     | 含义                       | 推荐展示           |
| -------- | -------------------------- | ------------------ |
| 待审核   | 商机提交后等待平台审核     | 发布方看到等待说明 |
| 招募中   | 商机可被服务商申请         | 展示申请入口       |
| 申请待审 | 服务商已申请，等待平台处理 | 申请方看到等待说明 |
| 已入围   | 申请方进入候选或推荐列表   | 展示后续对接说明   |
| 未入围   | 申请未通过                 | 展示结果说明       |
| 已撮合   | 平台完成撮合               | 双方展示对接结果   |
| 已关闭   | 发布方或平台关闭商机       | 不再允许申请       |

接口缺口：当前需要后台商机审核列表、申请列表、推荐服务商、撮合成功/失败和用户侧申请状态接口。
